# Both Hands V04: Cosmic Memory — Refined

## Purpose
This version refines V03 in response to tutor feedback. It preserves the successful cosmic-memory interaction while making the physical range clearer and the interface quieter.

## Changes from V03
- The field contracts to a much smaller centre when hands are close together.
- Its maximum radius is reduced so the full form remains inside the frame.
- The sketch opens with a dedicated help screen, so instructions do not remain over the artwork.
- `?` reopens or closes the help screen at any time.
- A quiet exhibition-style caption remains visible during the experience: the title is on the left and keyboard controls are on the right.
- `P` cycles through hand display modes: hidden, index-finger points, and full hand skeleton.

## Interaction
Move two hands apart to open the field. After reaching a wide stretch, bring them back together to leave one orbit-like memory ring. Each completed cycle places the next ring closer to the centre.

## Testing focus
- Can a visitor understand the small-to-large range without explanation?
- Does the skeleton mode help during testing without becoming part of the final visual?
- Is the quieter default screen more exhibition-like?

## Keys
- `M`: mouse / HandPose mode
- `?`: show / hide help
- `P`: hand display mode
- `R`: clear memory rings
